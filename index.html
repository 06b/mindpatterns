About

This site contains all working examples for the book eBay MIND Patterns - Accessibility Patterns for the Web.

Each example demonstrates the correct semantic HTML markup, keyboard interaction design, and ARIA attributes for assistive technology.

These examples are not the final, packaged code to be deployed or copy & pasted to your live customer-facing site! All patterns leave additional steps to complete in terms of the desired CSS styling and all JavaScript behaviour not related to accessibility (e.g. fit & finish, I18N/L10N, security, performance).
