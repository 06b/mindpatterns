Readme

Use left & right button controls to paginate the carousel slides. Use TAB key to navigate through contents of each slide.

Activate screen reader to hear carousel status updates (e.g. 'Showing slide 2 of 8').

Please pay no attention to the slide fade in and out animation (and it's glitches!). We use this animation for simplicity sake, but most likely a real carousel would utilise a scroll effect. Again, this is a timely reminder that our emphasis with these examples is accessibility concepts only.

Please read the documentation to understand how these examples were created.
