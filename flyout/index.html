Readme

Use ENTER or SPACEBAR to open flyout overlay, then use TAB key to move to the first interactive element or widget inside the overlay.

Flyout is an absract pattern. Concrete implementations include Tooltip, Bubble Help and Nav Menu.

Please read the documentation to understand how these examples were created.
